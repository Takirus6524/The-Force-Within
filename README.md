# THE FORCE WITHIN

A premium single-page training app for awareness, discipline, and emotional control.

## What this build includes

- Cinematic screen transitions for a smoother Core, Train, Reset, Schedule, Profile, and Emotion Log flow.
- A premium icon system for major actions and mission controls.
- A cleaner Philosophy experience with only one Philosophy button in the interface.
- Refined visual depth across cards, buttons, modals, and schedule items.
- Local persistence for trainer state, missions, schedule, and progress.

## Main Areas

- Intro and name setup
- Core dashboard
- Morning training flow
- Evening reset flow
- Schedule builder
- Emotion tracker
- Profile, statistics, achievements, reports, and timeline

## Notes

- The app is a single-file web experience centered in `index.html`.
- Data is stored locally in the browser.
- The Philosophy action now appears once from the Profile action panel.

## Validation

- JavaScript parsing was verified successfully.
- Browser checks confirmed the duplicate Philosophy button was removed and the premium icon treatment is active.
